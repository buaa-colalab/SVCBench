SVCBench A Streaming Video Counting Benchmark for Spatial-Temporal State Maintenance

Counting, reframed as a controlled probe for world-state maintenance. Rather than scoring only a final answer, SVCBench asks video-language models to update object and event counts along the timeline, exposing exactly when and where state tracking breaks.

Motivation

Counting is a compact stress test for world-state maintenance.

Long-video understanding is not just recognizing what happened. A model must continuously maintain internal state as the scene evolves: how many objects are visible now, how many unique identities have appeared, how many actions occurred, and how many full activity cycles have completed.

SVCBench decomposes this ability into object counting and event counting. Every video carries frame-level state-change annotations, then is queried at multiple timestamps — so evaluation can localize drift, missed updates, and inconsistent trajectories instead of grading a single final number.

Benchmark Design

Eight subcategories: snapshots, identities, events, and cycles.

Two families, four axes of difficulty, each probing a distinct failure mode of streaming state maintenance.

O1

Visible Object State

Snapshot and delta queries test whether counts update correctly as objects enter, leave, or become occluded.

Snapshot · Delta
O2

Cumulative Identity State

Unique-object and gain queries test whether identities are remembered after they disappear from view.

Unique · Gain
E1

Instantaneous Event State

Action and transit queries require detecting event-occurrence moments without static visual shortcuts.

Action · Transit
E2

Complete Activity Cycles

Episode and periodic queries measure whether the model tracks complete, repeating event cycles over time.

Episode · Periodic

Data

A timeline-first benchmark package.

The homepage should make the dataset feel inspectable before download. The final page can add short clips and GIFs here; this draft stays independent from unfinished camera-ready assets.

  • Video metadata and subcategory labels
  • Frame-by-frame object and event state annotations
  • Streaming query timestamps with ground-truth trajectories
  • Reference scoring scripts for GPA, MoC, and UDA
GPA

Gaussian Precision Accuracy

Rewards numerically close counts; tolerates near-misses with a soft Gaussian around the ground truth.

MoC

Monotonicity Consistency

Checks trajectory shape for cumulative quantities that should never decrease over time.

UDA

Update Detection Accuracy

Tests whether the prediction actually changes when the ground-truth state changes.
